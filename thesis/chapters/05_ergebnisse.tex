Der Studierende muss eine bestimmte Anzahl von Kursen wiederholen. Diese sind mit einer Priorität von 100 markiert und verletzen keine Hard Constraints. Abgesehen davon wurden keine Constraints gesetzt.

\begin{verbatim}
    {
        "FIXED_TIME_CONSTRAINTS": null,
        "COURSE_PRIORITY_CONSTRAINTS": {
            "5576": 100, # ADP
