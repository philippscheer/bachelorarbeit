\section{Datenbereitstellung}

\subsection{Erhebung der Vorlesungsdaten}

Die Datenextraktion der Vorlesungsdaten erfolgte über einen automatisierten Web-Scraping-Prozess, implementiert in Python 3.11 unter Verwendung der Bibliotheken \verb|requests| (HTTP-Requests), \verb|BeautifulSoup| (HTML-Parsing) und \verb|pandas| (Datenstrukturierung). Der Prozess umfasste folgende, drei Phasen:

\begin{enumerate}
    \item Extraktion relevanter Datenpunkte für alle Vorlesungen der Wirtschaftsuniversität Wien, 
    \item Identifikation relevanter Lehrveranstaltungen (Planpunkte) und 
    \item ECTS-Credits-Zuordnung gemäß Studien-Planpunkt.
\end{enumerate}


\subsubsection{Web-Scraping von Vorlesungsdaten}

Jede Lehrveranstaltung an der Wirtschaftsuniversität Wien wird durch eine eindeutige LV-ID identifiziert. LV-IDs werden durch eine Zahl von 0 bis 9999 dargestellt. Jede Lehrveranstaltung kann unter folgender URL mit Angabe des Semesters (Sommersemester 25 = \verb|"S25"|) abgerufen werden:

\begin{verbatim}
https://vvz.wu.ac.at/cgi-bin/vvz.pl?C=S&LANG=DE\
  &U=H&S=25S&LV=3&L2=S&L3=S&T=&L=&I=1337&JOIN=AND
\end{verbatim}

\hint{Beispiel: Abfrage der Lehrveranstaltung mit ID 1337 im Semester "25S".}


Aus der HTTP-Antwort jeder Anfrage wurden mit \verb|BeautifulSoup4| folgende Eigenschaften ausgelesen:

\begin{itemize}
    \item Zeitliche Parameter: Termine, Dauer
    \item Räumliche Parameter: Raumzuweisungen (regex-bereinigt)
    \item Personelle Zuordnung: LV-Leiter:innen
    \item Curriculare Verknüpfungen: Planpunkte-ID via RegEx \verb|r"P=([0-9]+);"| aus URL extrahiert
\end{itemize}

Für jede Planpunkt-ID erfolgte ein zweiter Scraping-Durchlauf, um die Zahl der ECTS Credits zu ermitteln.
Die finalen Daten wurden als pandas-DataFrame mit folgenden Spalten strukturiert:

\begin{itemize}
    \item \verb|id|: Eindeutige LV-Kennung
    \item \verb|dates|: Liste von Dicts mit Start/End-Zeitstempeln
    \item \verb|lvLeiter|: Verantwortliche Lehrperson
    \item \verb|planpunkte_ids|: Planpunkt-IDs
    \item \verb|ects|: Vom Planpunkt abgeleitete ECTS-Punkte
\end{itemize}
